Change boot time grub:
cd /etc/default
sudo cp grub gurb.bk
sudo vim /etc/default/grub
set grub timeout into <x>
ex :
GRUB_TIMEOUT=0
save
sudo update-grub2
restart

Disable display manager:
sudo systemctl disable gdm3.service

Disable splash screen (cp original file for backup):
sudo nvim /etc/default/grub
find the following line:
GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"
And change it to:
GRUB_CMDLINE_LINUX_DEFAULT=""
and then..
sudo update-grub2
sudo reboot

Change tty font (optional, for tty console font):
sudo dpkg-reconfigure console-setup
-ISO-8859-15
-Combined - Latin; Slavic Crtillic; Greek
-TerminusBold or Terminus or Fixed
-16x32 (Framebuffer only)
sudo update-initramfs -u
sudo update-grub2

source list:
sudo nvim /etc/apt/sources.list
add contrib non-free, in each last line
ex :
deb http://deb.debian.org/debian bullseye main contrib non-free
deb-src http://deb.debian.org/debian bullseye main contrib non-free

deb http://deb.debian.org/debian-security/ bullseye-security main contrib non-free
deb-src http://deb.debian.org/debian-security/ bullseye-security main contrib non-free

deb http://deb.debian.org/debian bullseye-updates main contrib non-free
deb-src http://deb.debian.org/debian bullseye-updates main contrib non-free

Right alt not working:
Open "Tweaks" (aka "Gnome Tweaks")
go to
Keyboard & Mouse > Additional Layout Options > Key to choose the 3rd level
and put a check mark on Right Alt never chooses 3rd level.
